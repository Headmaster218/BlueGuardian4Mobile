BlueGuardian

Protecting Our Waters through IoT and Mobile Technologies

About the App

BlueGuardian is a Flutter-based mobile application designed to monitor and visualize river water quality in real-time. Using IoT sensors and cloud communication, it empowers communities to protect and preserve aquatic environments.

Key Features

Real-Time Monitoring

Live data updates from river sensors via MQTT protocol.

Historical Data

View and analyze past water quality trends through dynamic charts.

Interactive Map

Explore river sites and sensor locations on an intuitive map interface.

Customizable Settings

Easily configure MQTT server settings within the app.
